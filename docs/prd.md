# 🧱 StackTap – Simple App Plan

## 1️⃣ App Goal

StackTap is a minimal arcade game where players tap the screen to stack moving blocks on top of each other.

The objective:
Stack as high as possible without misaligning blocks.

Simple. Addictive. Quick sessions.

---

## 2️⃣ What The App Does

* A block moves horizontally across the screen
* User taps to drop it
* If aligned with previous block → stack continues
* If partially aligned → block trims to overlap
* If no overlap → game over
* Score increases per successful stack
* High score saved locally

That’s it.

---

## 3️⃣ What The App Does NOT Do

* No multiplayer (v1)
* No in-game purchases (initially)
* No accounts
* No gambling mechanics
* No loot boxes
* No ads required at launch

Pure arcade skill game.

---

# 4️⃣ Core User Story

> As a player, I tap at the right moment to stack blocks, try to build the tallest tower possible, and attempt to beat my high score.

Fast, repeatable loop.
